AI Portfolio Website built as part of the Prompt Engineering Internship at Future Interns.

This project demonstrates the creation of a personal portfolio website using Durable AI, a no-code AI website builder. The website showcases professional skills, projects, and services while leveraging AI for content generation, layout design, and automated features.

Key Features:
- AI-generated homepage introduction
- AI-written about section and professional bio
- Skills & services section generated with AI
- Portfolio project summaries created using AI
- Built-in contact form with automated responses

Tools Used:
- Durable AI (No-Code AI Website Builder)
- Screenshot / video documentation included
